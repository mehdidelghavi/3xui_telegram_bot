
# ربات مدیریت و فروش X-ui

یک ربات تلگرام قدرتمند برای مدیریت و فروش پنل مبتنی بر XUI مانند:

- [x] 3X-UI

## نصب تک خطی

```bash
bash <(curl -Ls https://raw.githubusercontent.com/mehdidelghavi/3xui_telegram_bot/refs/heads/main/install.sh)
```
## مقادیر مورد نیاز برای نصب

قبل از نصب ربات بر روی سرور شما باید وارد تلگرام بشید و دو عدد ربات (یکی برای فروش و یکی برای مدیریت) از @botfather بسازید و api token آن هارو برای مراحل نصب داشته باشید

همچنین برای ذخیره رسید های کاربران و مدیریت مالی باید یک کانال Private بسازید و آیدی عددی کانال رو یادداشت کنید

بعد از انجام مراحل بالا باید هردو ربات را که ساخته بودید در این کانال ادمین کنید

تمامی موارد پایین الزامی میباشند
| Variable | Description |
|----------|------------|
|DB_NAME|نام دیتابیس|
|DB_USER|کاربر دیتابیس|
|DB_PASS|رمز عبور دیتابیس|
|Notebook_Channel_Id|آیدی عددی کانال برای ذخیره موارد مالی|
|BOT_TOKEN|توکن ربات تلگرام|
|ADMIN_CHAT_ID|ایدی عددی تلگرام برای ادمین|
|SELLBOT_TOKEN|توکن ربات فروش|
|ADMIN_USERNAME|نام کاربری برای پنل|
|ADMIN_PASSWORD|رمز عبور برای پنل|



# امکانات ربات

## ربات فروش 
- خرید / تمدید کانفیگ
- شارژ کیف پول
- نمایش حجم و زمان باقی مانده کانفیگ 
- نمایش کانفیگ های خریداری شده

## مدیریت کلاینت ها
- افزودن کلاینت [Service Based]
- ویرایش کلاینت
- تمدید و بروزرسانی کلاینت
- ریست کردن ترافیک کلاینت

## مدیریت سرویس ها
- افزودن سرویس جدید
- حذف سرویس
- Attach کردن اینباند به سرویس
- Detach کردن اینباند از سرویس

## مدیریت پنل / سرور ها
- افزودن پنل / سرور
- ویرایش پنل / سرور
- نمایش پنل / سرور
## دستورات مدیریتی

بررسی وضعیت ربات

```bash
pm2 status
```

نمایش لاگ ها

```bash
pm2 logs xui-bot
```

ریستارت ربات

```bash
pm2 restart xui-bot
```
